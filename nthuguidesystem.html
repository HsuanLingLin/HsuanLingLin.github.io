National Tsing Hua University Library Wireless Tour Guide

Developed a wireless tour guide system for NTHU library.

Project Background: Thesis Project

Project supervisor: Dr. Yuh-Ren Tsai

Abstract

A wireless tour guide system for NTHU Library by integrating embedded systems (controllers), Bluetooth USB dongles (tour nodes), and content management server.

Scenario: when users are in coverage of tour node, the tour guide app will show users’ location and provide them information of the area (e.g., Photo, introduction, audio, video…).

Architecture Design

  • Provided location-based service for users by placing at least one Linux-based embedded system (controllers) on every floor and number of Bluetooth USB dongles connected to the controller (Raspberry Pi) via USB HUB.
  • The program focused on managing controllers of every floor and tour nodes by implementing the integration of content management server and Linux-based embedded system via MQTT protocol (IoT protocol).

  • Designed MQTT topics/messages and scripts in Linux-based embedded systems to enable automatic configuration of embedded systems, and enable web application to monitor connections of guide nodes.
  • Created web application as GUI tool on server for admin to manage content of areas, initialize settings, monitor guide nodes and provide statistics results. With the monitoring feature, the admin will know the status of nodes (no response or alive), and statistics results feature will let the admin know hits of each file type. As a result, they will know how to improve user experience through statistic information.
