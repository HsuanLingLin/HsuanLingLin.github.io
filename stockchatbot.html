Real-time Stock Query bot

Project background:

side project

Overview:

Line chatbot with offering real-time stock price query. Leveraged the Flask framework, implemented database via Firebase, and deployed the webapp using Heroku. Scraping stock price with PyQuery.


Utilized:

Python Flask, Firebase(database), Ngrok(testing deployment), Heroku(deployment)

User Interface

  • Send the stock ID to get the real-time stock price.
